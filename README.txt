1. 라이센스

GR Board 는 GR Series 의 한 종류로서, 설치형 웹표준 기반 게시판 엔진입니다.
GPL v2 를 따르며 이는 여러분들이 자유롭게 GR Board 를 개선시키거나 수정하여
재배포 하실 수 있음을 뜻합니다. 단, 수정 후 재배포 시 소스코드를 모두 공개하셔야 하며
GR Board 를 사용함에 있어 문제가 발생했을 때 원작자는 책임을 질 수 없음을 밝힙니다.
코드는 공유하고, 즐거움은 함께 합시다.


2. 개발참여

GR Board 는 오픈소스 프로젝트로서 소스코드를 누구나 쉽게 내려 받아 수정하고
다시 이를 서버에 반영 하실 수 있도록 개방된 소스 관리를 하고 있습니다.
2011년 11월 부터는 기존에 사용하던 SVN 에서 Git 로 소스코드 관리 도구를 변경하였습니다.
Github 를 통해 제공되며 주소는 아래와 같습니다.

https://github.com/sirini/GR-Board

Git 사용법은 아래의 url 을 참조하시면 도움이 되실 겁니다.

http://namhyung.springnote.com/pages/3132772

GR Board 를 비롯한 GR Series 들의 경우 현재 시리니넷을 통해 배포가 진행되고 있습니다.
개발자 커뮤니티 사이트로서 주소는 아래와 같습니다.

http://sirini.net

개발은 코드를 기여해 주시는 것으로 참여해 주셔도 좋고, 버그를 제보해 주셔도 좋습니다.
문서화에 도움을 주셔도 되고, 홍보에 참여해 주셔도 좋습니다. 여러분들이 GR Board 와
함께 하시는 모든 것들이 개발에 참여하시는 것이 됩니다.


3. 개발목표

우리는 대한민국을 너머 전 세계에서 가장 빠르고, 안전하며 사용이 편리한 게시판 엔진을 만듭니다.
우리는 HTML5 그리고 그 이후의 웹 세상에서 여전히 혁신적이며, 새로운 도전을 이어나갑니다.
우리는 함께 도전하고 함께 나아갑니다.



*** GR Board 에 직/간접적으로 공헌해주신 모든 분들에게 최대한의 경의를 보냅니다. ***
