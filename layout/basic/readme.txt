사용하고자 하실 경우는 그렇게 하셔도 됩니다.

basic 레이아웃상에서 메뉴들은 가급적 영어로 되어 있으면
좋습니다. 물론 한글도 상관은 없습니다만 기본 골격만 제공하는
관계로 한글로 만들었을 때의 디자인이 고려되어 있지 않습니다.

레이아웃 매니저를 통해서 사이트를 보여줄 경우 GR보드내에
생성된 게시판들의 상단/하단 설정이 중요해집니다.
상단과 하단을 "보일 게시판 선택" 항목의 설명에서처럼 하시고
상단 파일과 상단 내용이 충돌나지 않도록 "상단 내용" 에
게시판 생성시 기본으로 적혀져 있는 것들은 모두 지워주시기
바랍니다. (하단 내용 역시 동일 합니다.)

처음부터 소스코드를 하드코딩해서 웹사이트를 구축하셔도 되지만
만약 그런 것이 부담 되신다면 레이아웃 매니저를 사용하고,
basic 레이아웃을 조금씩 수정해보면서 웹사이트를 구축하는 것도
하나의 방법이 될 수 있습니다. 초보자분들을 위해 더 편하고
더 좋은 레이아웃으로 거듭날 수 있도록 노력하겠습니다.
